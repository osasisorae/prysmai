# Prysm AI Landing Page — Design Brainstorm

## Context
Prysm AI is a real-time security monitor for AI agents using mechanistic interpretability. Tagline: "See through your AI." The brand metaphor is a prism splitting light to reveal hidden colors — making the invisible visible. Target audience: ML engineers, CTOs at Series A-C startups deploying AI agents.

## Idea 1: "Dark Observatory" — Cybersecurity Command Center

**Design Movement:** Dark-mode data visualization aesthetic inspired by Bloomberg Terminal meets NASA mission control. Think Vercel's dark elegance crossed with a security operations center.

**Core Principles:**
1. Authority through darkness — deep, rich blacks convey seriousness and technical credibility
2. Data as decoration — real-looking visualizations serve as both content and visual texture
3. Precision typography — monospace accents signal engineering culture
4. Controlled luminance — bright accents on dark backgrounds create focus hierarchy

**Color Philosophy:** Near-black base (#0A0B0F) with electric cyan (#00E5FF) as the primary accent — evoking a radar screen or security scanner. Secondary warm amber (#FFB800) for warnings/alerts. Sparse white for text. The palette communicates "we are watching, we are protecting."

**Layout Paradigm:** Full-bleed dark canvas with floating "panels" that resemble a monitoring dashboard. Hero section features an animated visualization of neural network activations. Content sections are stacked asymmetrically with generous negative space.

**Signature Elements:**
1. A live-looking "threat monitor" animation in the hero — glowing nodes and connections that pulse
2. Terminal-style code blocks showing `pip install prysm-middleware` with syntax highlighting
3. Subtle scan-line or grid overlay texture on backgrounds

**Interaction Philosophy:** Precise, mechanical interactions. Elements snap into place. Hover states reveal data. Scroll triggers sequential panel reveals like a mission briefing.

**Animation:** Entrance animations are fade-up with slight scale. The hero visualization has continuous subtle particle movement. Section transitions use staggered reveals. Code blocks type themselves out.

**Typography System:** JetBrains Mono for code/accents, Space Grotesk for headings, Inter for body. The monospace-sans pairing signals "engineering tool, not marketing fluff."

## Idea 2: "Light Refraction" — The Prism Metaphor Made Visual

**Design Movement:** Glassmorphism meets scientific illustration. Inspired by Apple's product pages crossed with physics textbook diagrams. Clean, light, with prismatic color refractions.

**Core Principles:**
1. The prism metaphor is literal — light splitting into spectrum is the visual motif
2. Clarity through whitespace — the product makes things clear, so should the design
3. Scientific credibility — clean diagrams and precise layouts signal rigor
4. Progressive disclosure — information unfolds like a research paper

**Color Philosophy:** Warm white base (#FAFAF8) with a rainbow spectrum gradient used sparingly as the signature accent — representing light being split by a prism. Primary interactive color is deep indigo (#4338CA). The spectrum appears in key moments: hero, dividers, hover states. It's not playful rainbow — it's physics.

**Layout Paradigm:** Wide, editorial layout with a strong vertical rhythm. Hero uses a large centered prism illustration with light beams splitting. Content alternates between full-width diagrams and two-column text+visual sections. Generous margins create a "published paper" feel.

**Signature Elements:**
1. A stylized prism illustration that splits a beam of "AI output" into colored feature streams
2. Thin rainbow-spectrum lines used as section dividers
3. Frosted glass cards with subtle backdrop blur for feature callouts

**Interaction Philosophy:** Smooth, optical interactions. Hover states create subtle prismatic color shifts. Scroll reveals content like turning pages of a research paper.

**Animation:** Smooth, physics-based easing. The hero prism animation shows light entering and splitting in real-time. Cards float up with spring physics. Spectrum lines draw themselves on scroll.

**Typography System:** Instrument Serif for the main headline (scientific authority), Instrument Sans for subheadings, system sans-serif for body. The serif headline is unexpected for a tech product and creates instant differentiation.

## Idea 3: "Signal & Noise" — Brutalist Data Aesthetic

**Design Movement:** Neo-brutalist web design meets information design. Inspired by Bloomberg Businessweek layouts, Stripe's documentation clarity, and Swiss design grids. Raw, confident, information-dense.

**Core Principles:**
1. Content is king — no decorative elements that don't communicate
2. Bold typographic hierarchy creates visual interest without images
3. High contrast creates urgency and authority
4. Structured chaos — intentional asymmetry within a rigid grid

**Color Philosophy:** Off-white (#F5F0EB) with jet black text and a single, vivid signal color — electric red (#FF3333) — used only for critical elements (CTAs, threat indicators, key stats). The red represents the "signal" that Prysm detects in the "noise." Everything else is grayscale.

**Layout Paradigm:** Strict 12-column grid with intentional breaks. Hero is a massive typographic statement taking up 80% of viewport. Content sections use newspaper-style multi-column layouts. Stats and data are presented in oversized type.

**Signature Elements:**
1. Oversized statistics displayed in massive type (e.g., "73%" in 200px font)
2. A thick black border/rule system that divides sections like a newspaper
3. Red "alert" accents that appear on hover and key CTAs

**Interaction Philosophy:** Direct and immediate. No unnecessary transitions. Hover states are bold color swaps. The page feels like a high-stakes briefing document.

**Animation:** Minimal but impactful. Numbers count up on scroll. Sections clip-reveal with hard edges. The CTA button pulses subtly like an alert indicator.

**Typography System:** Clash Display (bold geometric sans) for headlines, Space Mono for data/stats, Satoshi for body text. The contrast between geometric display and monospace creates visual tension that mirrors "signal vs noise."

## Selected Approach: Idea 1 — "Dark Observatory"

**Rationale:** For a security product targeting ML engineers and CTOs, the dark command-center aesthetic communicates exactly the right message: seriousness, technical depth, and protection. It also provides the best canvas for showcasing the data visualization capabilities that are core to Prysm AI's value proposition. The dark theme is standard in developer tooling (VS Code, terminal environments) and will feel native to our target audience. The animated hero visualization will be the most compelling way to demonstrate what "seeing through your AI" actually looks like.
